I’ve put together a simple Error Code Lookup Tool that lets you type in a fault code (like ID030 or ID291) or even just describe the issue (ABS, engine, tyre pressure, etc.). The search is fuzzy, so even partial matches will find what you’re looking for.

It shows clear causes, what to actually do next, and links straight to the exact page in the manual which will open in a new tab if you want— no more scrolling through 200+ pages trying to find it. There’s also a “More Info” section with extra insights and useful external references if you want to dig deeper.
